# 隐私政策 / Privacy Policy

**中文**

GitHub Contribution Arcade **不收集、不存储、不传输任何用户数据**。

- 扩展只在你主动打开的 GitHub 个人主页上运行，读取页面里**已经渲染出来的贡献图**（本就是公开信息）来生成游戏。
- 不发起任何网络请求，不使用任何第三方服务器，不使用 cookie、不写入任何本地存储。
- 所有运算都在你的浏览器本地完成，关闭页面即消失。

---

**English**

GitHub Contribution Arcade **does not collect, store, or transmit any user data**.

- It runs only on GitHub profile pages you open, reading the already-rendered
  contribution graph (public information) to build the games.
- It makes no network requests, uses no third-party servers, and sets no
  cookies or storage of any kind.
- Everything runs locally in your browser and is gone when you close the page.
